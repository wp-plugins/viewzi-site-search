Add Viewzi Site Search to your Site.

== Description ==

Viewzi Site Search for WordPress takes all the visual wizardry that is found on Viewzi.com and integrates it into your site. Allow your readers to search your accounts on Flickr, smugmug, Picasa, Youtube, Vimeo or Viddler as well as your local blog content. Present your blog data in an intuitive timeline, or allow your readers to search your posts/pages and comments in one shot. All this and more is possible with the VSS Plugin for WordPress

== Installation ==

1. Upload the Viewzi Site Search folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to http://www.viewzi.com/profile/vss.php?force_login=1&action=create and create an account.
1. Access the Plugin management page, and enter your VSS ID (created in step 3), then refresh the management page.
1. Setup the views you want to run on your site.
1. Add the javascript snippet provided in your theme where ever you want the VSS search box to show up.
1. Profit!
